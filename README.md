# 智慧城市 demo

## 概述

    本项目是一个智慧城市的例子，主要使用技术svg解析，模型加载，辉光(bloom),纹理移动动画

![img](assets/example.png)

## 安装使用

    下载项目，或者git clone
    下载完成打开目录，安装包

    cnpm install 使用国内镜像要快一点
    or
    npm install

    安装完成后运行即可
    npm run dev
